About the Project

The AI Disease Prediction System is a web-based platform that allows users to input their symptoms and receive real-time predictions of possible diseases using Machine Learning. This project aims to promote early diagnosis, awareness, and accessible healthcare through AI technology.

🤖
AI Predictions

Symptom-based prediction using trained ML model.

User authentication & session management for safety.

🎨
Modern UI

Bootstrap-styled responsive interface for all devices.

🚀
Future Ready

Expandable for reports, chatbot, and doctor suggestions.
