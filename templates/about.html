About the Project

The AI Disease Prediction System is a web-based tool that allows users to select symptoms and get real-time predictions of possible diseases using Machine Learning. It is designed to assist users in early diagnosis and awareness, using a secure, simple interface.

🔍 Key Features

  • Symptom-based prediction using trained ML model
  • Secure user login and session management
  • User-friendly interface with Bootstrap styling
  • Expandable for future features like report download, chatbot, and specialist recommendation
